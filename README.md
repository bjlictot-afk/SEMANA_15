# 📝 Aplicación Lista de Tareas (To-Do List)

## 📌 Descripción

Aplicación de escritorio desarrollada en Python con Tkinter que permite gestionar tareas diarias. El usuario puede agregar, completar y eliminar tareas de forma interactiva.

---

## 🎯 Objetivo

Implementar una aplicación GUI aplicando arquitectura modular por capas, manejo de eventos y generación de un ejecutable.

---

## 🏗️ Estructura del Proyecto

```
lista_tareas_app/
│
├── main.py
├── modelos/
│   └── tarea.py
├── servicios/
│   └── tarea_servicio.py
├── ui/
│   └── app_tkinter.py
├── dist/
│   └── ListaTareas.exe
```

---

## ⚙️ Funcionalidades

* Agregar tareas
* Marcar tareas como completadas
* Eliminar tareas
* Visualización en lista
* Cambio visual al completar tareas

---

## 🎮 Manejo de Eventos

* Tecla Enter: agrega una tarea
* Doble clic: marca tarea como completada

---

## ▶️ Ejecución

Ejecutar desde Python:
python main.py

Ejecutar como aplicación:
dist/ListaTareas.exe

---

## 📦 Tecnologías

* Python
* Tkinter
